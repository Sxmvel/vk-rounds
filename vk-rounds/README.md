# 🥊 Organizador de Rounds - VK Team

Sistema inteligente para gestão de sparrings e rodízio de atletas. Desenvolvido para otimizar o tempo de treino e garantir confrontos justos.

## 🚀 Tecnologias
- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS + Ant Design
- **Algoritmo:** Weighted Round-robin (Matching Engine)
- **Deploy:** Vercel

## ⚖️ Regras de Negócio
- [x] Separação por categoria (Professores vs Alunos) nos rounds iniciais.
- [x] Rodízio justo para número ímpar de atletas (descanso equilibrado).
- [x] Minimização de repetição de oponentes.
- [x] Histórico dinâmico de lutas por round.
